# react-native-audio-bitstream

Audio bitstream sent to JS

## Installation

```sh
npm install react-native-audio-bitstream
```

## Usage

```js
import { multiply } from "react-native-audio-bitstream";

// ...

const result = await multiply(3, 7);
```
